# Contributing

This repo is primarily a portfolio project, but suggestions are welcome.

## Ways to contribute
- Improve layout/UX (spacing, alignment, typography)
- Add accessibility improvements (color contrast, label clarity)
- Add additional pages (e.g., Profitability, Geography)
- Provide alternative datasets / parameterized template version

## How to propose changes
1. Fork the repo
2. Create a feature branch
3. Submit a PR with a short explanation and screenshots
